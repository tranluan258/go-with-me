# Go + Echo + Websocket + HTMX
A chat application in Go enables users to log in with social accounts and engage in real-time messaging with others.

## Tool used
  - Go + Echo
  - WebSocket
  - HTMX
  - UI: TailwindCSS, DaisyUI
  - PostgresSQL
